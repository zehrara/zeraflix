# Zeraflix REST API

**API Adresi:** https://zeraflix.vercel.app

## Endpointler

| Method | Endpoint | Açıklama |
|--------|----------|----------|
| POST | /api/register | Kullanıcı kaydı |
| POST | /api/login | Giriş yapma |
| PUT | /api/profile | Profil güncelleme |
| DELETE | /api/profile | Profil silme |
| GET | /api/content | Film/dizi listeleme |
| GET | /api/search | İçerik arama |
| POST | /api/watchlist | Listeye ekleme |
| DELETE | /api/watchlist | Listeden çıkarma |
| PUT | /api/rate | Puan verme |
| GET | /api/history | İzleme geçmişi |
| GET | /api/recommendations | Kişisel öneri |
